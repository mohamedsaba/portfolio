# Visible Queues

What you can't measure, you can't manage. BullMQ Metrics makes it trivial to track queue health, job latency, and failure rates.

## Prometheus Integration

The library automatically sets up a Prometheus exporter that tracks:
- `bullmq_jobs_waiting`
- `bullmq_jobs_active`
- `bullmq_job_duration_seconds`

```javascript
import { createMetricsCollector } from '@mohamedsaba/bullmq-metrics';

createMetricsCollector(myQueue, {
  port: 9091,
  endpoint: '/metrics'
});
```
